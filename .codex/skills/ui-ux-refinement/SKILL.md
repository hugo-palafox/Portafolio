# UI/UX Refinement

## Use this skill when
- Building or refactoring page layout, section structure, or visual hierarchy.
- Adjusting card grids, metadata placement, or content density.

## Rules
- Prefer whitespace and scanability over dense interfaces.
- Keep hierarchy clear: hero -> section -> card -> metadata.
- Align sections and cards to shared grid columns.
- Keep dark theme readable with strong contrast.
- Remove unnecessary chrome and decorative clutter.
- Favor clarity and readability over visual effects.

## Validation checklist
- Is spacing rhythm consistent section-to-section?
- Is information hierarchy clear in 5 seconds?
- Are mobile and desktop layouts both balanced?
- Is the page still portfolio-first, not dashboard-like?
