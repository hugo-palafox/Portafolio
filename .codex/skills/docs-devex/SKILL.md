# Documentation and Developer Experience

## Use this skill when
- Adding features that require usage guidance or architecture notes.
- Finalizing milestones and implementation summaries.

## Responsibilities
- Keep README concise and current.
- Document non-obvious architectural decisions.
- Add component usage notes for reusable UI patterns.
- Keep deployment instructions accurate.

## Deliverables
- README updates
- usage documentation
- component usage notes
- deployment instructions
- changelog/progress updates when applicable
