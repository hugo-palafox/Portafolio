# Architecture Diagram Generation

## Use this skill when
- Explaining system layers, integrations, service boundaries, or data flow.

## Preferred formats
- Mermaid
- draw.io compatible diagrams
- simple architecture block diagrams

## Rules
- Keep diagrams focused on system understanding.
- Label interfaces and flow direction clearly.
- Match diagrams to implemented behavior.
- Avoid decorative complexity.
